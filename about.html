About This Project

The Inventory Command Center was built for ABC Logistics to provide the warehouse team with a simple, fast, and reliable way to track inventory. Key features include:


✓ Real-time item tracking with add, delete, and clear functionality
✓ Visual analytics with toggleable Bar and Pie charts
✓ Persistent data storage using browser localStorage
✓ Input validation to prevent errors and duplicates
✓ Responsive design that works on desktop and mobile
